- for the time being, work against the following locations:
  Storage bucket: us
  project config

* Create a project in Google Cloud (see https://cloud.google.com/functions/docs/quickstart)
* Enable billing (see https://cloud.google.com/functions/docs/quickstart)
* Enable Google Cloud Functions (see https://cloud.google.com/functions/docs/quickstart)

* create gcloud project config with ./commands/create_gcloud_configuration.sh 

* create a Cloud Storage bucket with appropriate name
* deploy cloud function via bucket



* For local dev, install the Cloud Functions emulator ( https://cloud.google.com/functions/docs/emulator )
  - install nvm
	follow instructions at https://github.com/creationix/nvm
  - install & switch to nodejs 6.9.1
	nvm install 6.9.1
  - install cloud functions emulator
	npm install @google-cloud/functions-emulator
